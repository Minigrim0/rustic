Audio subsystem with three-thread architecture for real-time safe audio Audio subsystem with three-thread architecture

This module implements a real-time safe audio architecture:

  • Command thread: validates commands and updates state
  • Render thread: generates audio samples
  • cpal callback: copies pre-rendered audio to output

The architecture uses lock-free ring buffers for communication and ensures the cpal callback has minimal work to do.
